# Escape Room 2.0

A terminal-based escape room game built using Python.

## Description

Escape Room 2.0 is a text-based adventure game where the player must survive four different rooms and escape the dungeon. The player begins with **100 health** and an empty inventory. Each decision affects the player's health, inventory, and the final outcome of the game.

## Features

* Four interactive rooms
* Health management system
* Inventory system
* Random events
* Multiple player choices
* Win and lose conditions
* Modular program structure using functions

## Concepts Used

* Functions
* Variables
* Lists
* Conditional statements (`if`, `elif`, `else`)
* User input
* Random module
* Function parameters and return values

## How to Run

1. Ensure Python 3 is installed.
2. Clone or download this repository.
3. Open a terminal in the project folder.
4. Run the program:

```bash
python escape_room.py
```
