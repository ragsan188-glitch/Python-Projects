# 🏃 Escape Room Game (Python)

A beginner-friendly text-based Escape Room game built using Python.

The player must solve puzzles behind different doors while managing limited lives. Correct answers reward badges, while incorrect answers reduce lives.

---

## 🎮 Features

- 🧮 Math puzzle
- 🔍 Mystery word puzzle
- 🎲 Random lucky number game
- ❤️ 3 lives system
- 🏅 Badge collection using Python sets
- 🚪 Menu-driven gameplay

---

## Concepts Used

- Functions
- Loops
- Conditional statements
- User input
- Random module
- Sets
- Function parameters
- Return values

---

## How to Run

1. Install Python 3.
2. Download or clone this repository.
3. Open a terminal.
4. Run:

```bash
python escape_room.py
```

---

## Game Rules

- You start with **3 lives**.
- Choose one of the three doors.
- Solve the puzzle behind the door.
- Correct answers earn badges.
- Wrong answers reduce one life.
- The game ends when:
  - You lose all your lives, or
  - You choose option **4** to exit.
