# Number Guessing Game

## Description
A beginner Python project where the computer randomly selects a number between 1 and 20, and the player has 5 attempts to guess it.

## Concepts Used
- Variables
- Functions
- While loops
- Conditional statements
- User input
- Random module
- Break statement

##What I Learned
- Creating functions
- Using loops effectively
- Handling user input
- Implementing game logic
- Debugging and improving code

## How to Run
```bash
python number_guessing_game.py
```
